# Prompts Guidelines

## Prime directive

Prompts must force evidence discipline, not bypass it.

## Required prompt behavior

Every serious VA prompt should require:

```text
read README.md
read AGENTS.md
read OVERVIEW_INDEX.md
read relevant folder INDEX.md
read relevant folder GUIDELINES.md
inspect current source before patching
patch existing owner only
update indexes when files change
report proof level honestly
```

## Forbidden prompt behavior

Prompts must not ask agents to:

```text
create V2 systems
skip owner inspection
rewrite broadly
invent duplicate FileIO/path/timer owners
claim compile/runtime proof without evidence
turn Trader Chat Pack into auto trading permission
use Gateway as hidden execution authority
```

## Index rule

Every prompt file must be listed in `prompts/INDEX.md` in the same patch.
