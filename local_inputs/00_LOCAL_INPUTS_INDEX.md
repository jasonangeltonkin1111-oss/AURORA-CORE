# 00 LOCAL INPUTS INDEX

Local staging input index.

- `.gitkeep` keeps the folder in repo.

## Folder Governance
- **Purpose:** temporary staging for local/manual inputs before governed promotion.
- **What belongs here:** transient staging artifacts only.
- **What must not belong here:** active source truth, runtime outputs, or permanent doctrine.
- **Authority boundary:** artifacts become authoritative only after promotion into governed folders.
- **Update rules:** keep this index minimal and current if staging conventions change.
- **No-go rules:** do not treat local staging files as implementation proof.
- **Relationship to master index:** this folder is listed in `control/02_MASTER_REPO_FILE_INDEX.md`.
