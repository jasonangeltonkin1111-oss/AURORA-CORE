# Research Guidelines

## Prime directive

Research supports design decisions. It does not override current source/config/runtime evidence.

## Source priority

```text
official documentation / primary sources
broker and prop-firm rule pages
verified runtime outputs
reputable technical references
trading research only as hypothesis
```

## Forbidden

Research must not:

```text
be used as auto-trading permission
claim edge proof without validation
turn social-media concepts into runtime truth
hide uncertainty
contradict MT5/Gateway ownership laws
```

## Index rule

Every research item or subfolder must be listed in `research/INDEX.md`.

## Status labels

Use:

```text
REFERENCE
DRAFT
CANDIDATE
REJECTED
SUPERSEDED
```
